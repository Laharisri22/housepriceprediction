# 🏠 House Price Prediction

This project uses **Machine Learning techniques to predict house prices** based on various features such as location, area, number of bedrooms, and other property-related factors. The model analyzes historical housing data to estimate the price of a house.

---

## 📊 Project Overview

House price prediction is an important application of **data science and machine learning**. This project analyzes housing data and builds a predictive model that estimates house prices using different property features.

---

## 🚀 Features

* 📂 Data cleaning and preprocessing
* 📊 Exploratory Data Analysis (EDA)
* 📈 Data visualization using graphs and charts
* 🤖 Machine Learning model for price prediction
* 📉 Model evaluation using performance metrics

---

## 🛠️ Technologies Used

* **Python**
* **Pandas** – Data analysis and manipulation
* **NumPy** – Numerical computations
* **Matplotlib** – Data visualization
* **Seaborn** – Statistical visualization
* **Scikit-learn** – Machine learning algorithms
* **Jupyter Notebook** – Project development

---

## 📂 Dataset

The dataset contains information about houses such as:

* Area (Square Feet)
* Number of Bedrooms
* Number of Bathrooms
* Location
* Year Built
* Price

These features are used to train the machine learning model to predict house prices.

---

## 🤖 Machine Learning Model

The project uses machine learning algorithms such as:

* **Linear Regression**
* **Decision Tree Regression**
* **Random Forest Regression**

These models learn from the dataset and predict house prices based on input features.

---

## 📊 Model Evaluation

The model performance is evaluated using:

* **R² Score**
* **Mean Squared Error (MSE)**
* **Mean Absolute Error (MAE)**

## ▶️ How to Run the Project

1. Clone the repository

```

```

2. Open the project folder.

3. Install required libraries

```

```

4. Run the Jupyter Notebook

```

```

5. Open **house_price_analysis.ipynb** to view the analysis and predictions.
